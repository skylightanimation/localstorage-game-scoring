# localstorage-game-scoring

**TouchSpeed Meter**, game to count your click speed <desktop device> or touch speed <mobile device> at 60 seconds.
This game was build to try **How localStorage is works** for tutorial [Web Storage on the Client Side (Local Storage and Session Storage)](https://langittutorial.blogspot.com/2020/08/mengenal-memahami-web-storage-pada-client-side-dan-cara-menggunakan-session-storage-dan-local-storage.html).

Try : [Run in Browser](https://skylightanimation.itch.io/touchspeed-meter)
